Colchicine for pericarditis
=================================

This repository contains the [source data](https://github.com/openMetaAnalysis/Colchicine-for-pericarditis/tree/master/data) and matching [forest plots](https://github.com/openMetaAnalysis/Vitamin-D-for-fracture-prevention/tree/master/forest%20plots) for an [open  meta-analysis](https://public.opencpu.org/ocpu/github/openMetaAnalysis/binary/www/) of colchicine for pericarditis.

This meta-analysis suggests that colchicine reduces the rates of recurrence of pericarditis.
